static page generator


how to use

  1. fill out the form with the info you want on your website
  2. click "generate!", and a file named "index.html" will be downloaded containing the code. put it in a folder somewhere on your computer (like your Documents folder)
  3. download this file and put it in the same folder: style.css
  4. put whatever image you'd like to be your icon into the folder as a 512x512 png file named icon.png (like this one)
  5. upload all of the files to neocities or wherever you like to put your static websites
